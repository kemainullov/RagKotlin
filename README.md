# RagKotlin

RAG-ассистент разработчика на Kotlin. Индексирует документацию проекта, ищет релевантные фрагменты через эмбеддинги и генерирует ответы с помощью LLM.

## Зависимости

- **Ollama** — локальный сервер для генерации эмбеддингов (модель `nomic-embed-text`)
- **DeepSeek API** — LLM для генерации ответов и реранкинга
- **JDK 24** — среда выполнения
- **Gradle** — система сборки

## Быстрый старт

1. Запустите Ollama: `ollama serve`
2. Укажите API-ключ DeepSeek в `local.properties`:
   ```
   DEEPSEEK_API_KEY=sk-...
   ```
3. Запустите приложение:
   ```bash
   ./gradlew run
   ```

## Команды чата

| Команда | Описание |
|---------|----------|
| `/help <вопрос>` | Поиск по документации проекта через RAG |
| `/index` | Переиндексация документов |
| `очистить` | Сброс истории диалога |
| `выход` | Завершение работы |

## Структура проекта

```
├── src/main/kotlin/Main.kt  — основная логика (клиенты, индекс, RAG, чат)
├── data/                     — текстовые документы для индексации
├── docs/                     — документация проекта
│   ├── architecture.md       — архитектура
│   └── code-style.md         — правила стиля кода
├── index.json                — индекс эмбеддингов (генерируется автоматически)
├── build.gradle.kts          — конфигурация сборки
└── local.properties          — локальные настройки (API-ключи)
```
