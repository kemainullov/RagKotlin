# Release Notes v1.1

Версия v1.1 представляет собой значительное расширение функциональности проекта за счёт интеграции новых ассистентов и автоматизации процессов. Основные изменения сосредоточены на добавлении возможностей для командной работы и автоматического анализа кода, что улучшает рабочий процесс разработки.

## Features
*   Добавлен ассистент для работы с пользователями.
*   Добавлен ассистент для командной работы.

## Improvements
*   Добавлен рабочий процесс для автоматического ревью кода с использованием ИИ.
*   Добавлена функция для получения списка файлов в pull request.

## Other
*   Обновлён рабочий процесс автоматического ревью кода (версия 2).
